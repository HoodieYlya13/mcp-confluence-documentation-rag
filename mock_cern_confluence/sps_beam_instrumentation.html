SPS Beam Instrumentation VME Crate & Register Configurations

The SPS beam position monitoring (BPM) system relies on legacy VMEbus hardware located in BA3, BA4, and BA5. These systems capture orbit data at a sampling frequency of 40 MHz and must be configured via the primary timing registers.

1. VME Crate Memory Maps

The direct physical memory addresses for the beam loss monitors (BLM) and beam position monitors (BPM) controllers are specified below.

Crate ID System Type Base Register Address Interrupt Level DMA Channel Max Calibration Error
VME_SPS_BA3_BPM BPM Orbit Controller 0xFC000000 5 1 0.02 mm
VME_SPS_BA4_BLM Beam Loss Integrator 0xFD100000 4 2 0.05 Gy/s
VME_SPS_BA5_BCT Beam Current Transformer 0xFE200000 6 0 0.1% intensity

2. Hardware Calibration Procedure

  1. Initiate hardware reset using command word 0x00FF written directly to base + 0x08.
  2. Wait 200 ms for FPGA clock phase lock loops (PLL) to stabilize.
  3. Check register 0x00A4 for the lock bit (bit 12). If zero, calibration failed.

Performing calibration while beam is present will cause a false Beam Loss monitor trigger, resulting in an immediate beam abort. Calibration must only be run during dedicated machine development (MD) sessions.
